# 💧 Proyecto: Análisis de Calidad del Agua en Facatativá

## 📌 Descripción
Este proyecto tiene como objetivo analizar la calidad del agua en diferentes puntos de Facatativá utilizando herramientas de Python, como pandas y métodos numéricos.

Se evaluarán variables como:
- pH
- Oxígeno disuelto
- Conductividad
- Dureza
- Temperatura

---

## 🎯 Objetivos

### Objetivo general
Analizar la calidad del agua en distintos puntos de Facatativá mediante técnicas de análisis de datos.

### Objetivos específicos
- Explorar los datos recolectados
- Limpiar y procesar la información
- Clasificar la calidad del agua
- Aplicar un método numérico para análisis

## 👥 Integrantes y tareas

### 👤 Integrante 1 – Exploración
- Cargar el archivo CSV

### 👤 Integrante 2 – Procesamiento
- Limpiar datos
